# Policy — assessment integrity

1. Evidence-graded, always. No score of 2+ without inspection or sampling. If access is not granted, the cell caps at 1 and the access gap is itself a finding.
2. Fixed sample floors. 20 records per class (100% if fewer). Samples are chosen by the assessor, never supplied by the client.
3. Reference data is never skipped. It is the foundation class; a weak score there caps confidence in every other class and is reported first.
4. The client owns the findings. Written to stand alone — no phrasing that needs Zest in the room to interpret.
5. Scores are re-derivable. Every 2+ cites its evidence: screenshot ref, export ref, or sample id list.
6. Re-assess after each delivery wave or annually, whichever comes first. Same instrument version where possible; version drift is recorded on the scorecard.
7. Honesty over the sale. If the honest roadmap is "fix reference data, then stop and re-assess", that is the roadmap.
